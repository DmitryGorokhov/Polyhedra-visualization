Программа разбиения сферы на равные участки
    Программа предназначена для визуализации возможности
разбиения сферы на равные участки с помощью многогранников.
    Целью данной программы является визуализация многогранников,
наиболее подходящих для аппроксимации сферы. Программа
демонстрирует икосаэдр, додекаэдр и тетракисгексаэдр, а также
разбиение икосаэдра с помощью методов бисекции и трисекции. Для
каждой фигуры реализовано два варианта визуализации: показ вершин
фигуры, отображение реберной модели фигуры.
    Программа реализована на языке программирования Python 3.7.
Графический интерфейс создан с помощью библиотеки tkinter. Для
визуализации использовались библиотеки numpy и matplotlib.
